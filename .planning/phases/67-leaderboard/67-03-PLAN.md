**1. `LeaderboardPodium.tsx` — Top 3 visual treatment:**
- Props: `{ entries: LeaderboardEntry[] }` (expects up to 3 entries, already rank-sorted)
- Layout: horizontal flex with podium order **2nd - 1st - 3rd** (center elevated)
- Each podium card:
  - Background color tint: Gold `#B8860B` for 1st, Silver `#9A9A9A` for 2nd, Bronze `#8B4513` for 3rd — use at ~15% opacity as card background
  - Rank label at top (medal emoji or number with gold/silver/bronze color)
  - Avatar: `<Avatar name={entry.username} size={40} />` (larger than row avatars)
  - Username (Lora, 13px, C.ink, truncate at container width — `overflow: hidden; textOverflow: ellipsis; whiteSpace: nowrap; maxWidth: '90px'`)
  - Level (Bebas Neue, 11px, C.muted, "LV {level}")
  - XP (Bebas Neue, 13px, `#E8A020`, `{xp.toLocaleString()}`)
  - Tier badge dot: 6px colored circle positioned at top-right of avatar
- Card dimensions: flexible width with `flex: 1`, `maxWidth: '110px'`, `minWidth: '80px'`
- 1st place card: slightly taller — use `marginTop: '-12px'` or `paddingTop` to make the center column elevated
- Entrance animation: Framer Motion `motion.div` with `initial={{ y: 20, opacity: 0 }}` `animate={{ y: 0, opacity: 1 }}` and 0.1s stagger (index * 0.1 delay)
- Respect `useReducedMotion` from framer-motion — skip animation if user prefers reduced motion
- If fewer than 3 entries exist (e.g., only 1 or 2 players), render only what's available. Do not show empty podium slots.
- Container has `marginBottom: '24px'`, centered with `justifyContent: 'center'`, `gap: '12px'`

**2. `LeaderboardStickyYou.tsx` — Your rank row:**
- Props: `{ userRank: UserRank | null; isAuthenticated: boolean; isInTop25: boolean }`
- **If !isAuthenticated:** Show "Sign in to see your rank" with a Link to /login. Style: Lora italic, 14px, C.muted. Include a subtle arrow or "SIGN IN" button in Bebas Neue.
- **If isAuthenticated && isInTop25:** Render nothing (the user's row is already highlighted in the main list with accent background)
- **If isAuthenticated && !isInTop25 && userRank:** Show the user's rank row:
  - Same layout as LeaderboardRow but with separator above (2px solid C.rule with 16px margin-top)
  - Left side: "YOU" label in Bebas Neue, 12px, C.accent (instead of rank number)
  - Show: rank number, username, level, total_xp
  - Below the row: gap message "X XP to move up" in Lora italic, 12px, C.muted — only if gap_to_next > 0
- **If isAuthenticated && !isInTop25 && !userRank:** Show "Play a game to appear on the leaderboard!" with link to /play
- Position: This is NOT `position: sticky`. Instead, render it as the last element in the page content, below the ranked list, with a visual separator. The research suggested `position: sticky; bottom: 0` but that requires the list to be in a scroll container — since this is a page-level component with only 25 rows, a fixed bottom section is cleaner and avoids sticky parent issues.
