**Interface:**
```typescript
export interface FlaggedQuestionRow {
  id: number;
  externalId: string;
  text: string;
  difficulty: string;
  status: string;
  flagCount: number;
  collectionNames: string[];
  reasonBreakdown: Record<string, number>;  // e.g. { 'confusing-wording': 3, 'wrong-answer': 2 }
  lastFlaggedAt: string;
}

interface FlaggedQuestionsTableProps {
  questions: FlaggedQuestionRow[] | null;
  sort: string;
  order: string;
  onSortChange: (column: string) => void;
  onQuestionClick: (id: number) => void;
}
```

**Columns (in order):**
1. **Flags** — flag count as bold number, red if > 5, orange if > 2, gray otherwise. Sortable (column key: 'flag_count').
2. **Question** — truncated text (max-w-md truncate), same as QuestionTable.
3. **Collection** — collectionNames.join(', '), same as QuestionTable.
4. **Reasons** — Render reason breakdown as colored chips. Map each reason to a short label + color:
   - 'confusing-wording' => label "Confusing", class "bg-amber-100 text-amber-800"
   - 'outdated-info' => label "Outdated", class "bg-blue-100 text-blue-800"
   - 'wrong-answer' => label "Wrong", class "bg-red-100 text-red-800"
   - 'not-interesting' => label "Boring", class "bg-gray-100 text-gray-800"
   - Display as: `<span className="px-2 py-0.5 text-xs font-medium rounded-full {color}">{label} x{count}</span>`
   - Wrap in `flex flex-wrap gap-1`
   - If reasonBreakdown is empty/null, show dash "-"
5. **Last Flagged** — formatted date from lastFlaggedAt using `new Date(lastFlaggedAt).toLocaleDateString()`. Sortable (column key: 'created_at').

**Table styling:** Follow QuestionTable exactly:
- `<table className="min-w-full divide-y divide-gray-200">`

- Sortable headers have `cursor-pointer hover:bg-red-800`
- `renderSortIcon(columnKey)` shows arrow indicator
- `<tbody className="bg-white divide-y divide-gray-200">`
- Rows: `hover:bg-gray-50 cursor-pointer transition-colors`
- onClick calls `onQuestionClick(question.id)`

**Loading skeleton:** When questions is null, render 10 skeleton rows with `animate-pulse` and gray div placeholders (same pattern as QuestionTable).
