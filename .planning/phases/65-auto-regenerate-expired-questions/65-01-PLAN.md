**Exported function: `generateReplacement()`**

Signature:
```typescript
export async function generateReplacement(
  archivedQuestionId: number,
  archivedExternalId: string,
  topic: string | null,       // subcategory slug — nullable!
  collectionSlug: string,
  collectionId: number,
): Promise<ReplacementResult>
```

The ENTIRE function body is wrapped in try/catch. The catch block returns `{ replaced: false, reason: err.message }` — it NEVER throws to the caller. This mirrors the `awardPlatformXp` pattern from `progressionService.ts`.

**Internal flow (in order):**

1. **Null topic guard**: If `topic` is null, return `{ replaced: false, reason: 'no-topic' }`. Some old questions have null subcategory.

2. **Load locale config**: Call internal `tryLoadLocaleConfig(collectionSlug)`. If null, return `{ replaced: false, reason: 'no-locale-config' }`.

3. **Generate single question via Anthropic API**:
   - Import `client` and `MODEL` from `../scripts/content-generation/anthropic-client.js`
   - Import `buildSystemPrompt` from `../scripts/content-generation/prompts/system-prompt.js`
   - Build system prompt with `buildSystemPrompt(config.name, { [topic]: 1 }, config.collectionSlug, config.officeholders)` — this scopes generation to exactly 1 question in the target topic
   - Build user message: `"Generate exactly 1 civic trivia question in the '${topic}' topic for ${config.name}. Return JSON with a single 'questions' array containing 1 question. Use the externalId prefix '${config.externalIdPrefix}'."`
   - Call `client.messages.create({ model: MODEL, max_tokens: 2000, system: systemPrompt, messages: [{ role: 'user', content: userMessage }] })`
   - Extract text content from response, parse JSON

4. **Parse with QuestionSchema** (NOT BatchSchema — BatchSchema requires 15-30 questions and will throw):
   - Parse the JSON response. The AI returns `{ questions: [...] }` format. Extract `response.questions[0]`.
   - Validate with `QuestionSchema.parse(question)`. If parse fails on first attempt, retry generation ONCE (goto step 3). On second parse failure, return `{ replaced: false, reason: 'parse-error' }`.

5. **Run quality rules**:
   - Import `auditQuestion` from `../services/qualityRules/index.js`
   - Call `auditQuestion(parsedQuestion)` — check for blocking violations only (advisory violations are OK)
   - If blocking violations exist: return `{ replaced: false, reason: 'quality-fail: ' + violations.join(', ') }`. Do NOT retry on quality fail — the context says "cleanliness over gap-filling".

6. **Allocate externalId** using `getNextExternalId()` pattern (internal function — see below):
   - Query MAX numeric suffix from questions with matching prefix in the collection (include ALL statuses — archived IDs are still in DB with UNIQUE constraint)
   - Set `parsedQuestion.externalId = prefix + '-' + String(nextId).padStart(3, '0')`

7. **Semantic dedup check**:
   - Guard: if `!process.env.OPENAI_API_KEY`, skip dedup, log structured warn, proceed to seed
   - Dynamic import `OpenAIEmbeddingService` and `SemanticDupDetector`
   - Embed the replacement question text using `SemanticDupDetector.prepareTextForEmbedding()`
   - Query ACTIVE questions in the collection (WHERE status = 'active') — the just-archived question is excluded because its status is now 'expired'
   - Embed active questions (uses disk cache at `.embedding-cache/`)
   - Compute cosine similarity. If any score > 0.85: retry generation ONCE (goto step 3). On second near-dup hit, return `{ replaced: false, reason: 'near-duplicate' }`.

8. **Resolve topicId**:
   - Query `topics` table for the topic slug: `db.select({ id: topics.id }).from(topics).where(eq(topics.slug, topic))`
   - If no topic found, return `{ replaced: false, reason: 'topic-not-found' }`

9. **Seed as active** (direct insert, NOT via seedQuestionBatch which hardcodes 'draft'):
   - Insert into `questions` table with ALL fields:
     ```
     externalId, text, options, correctAnswer, explanation, difficulty, topicId, subcategory: topic,
     source, learningContent: null, expiresAt (from parsed question, may be non-null for officeholder questions),
     status: 'active' as const, expirationHistory: []
     ```
   - Use `.onConflictDoNothing()` on insert (safety net for externalId collision)
   - If insert returns no rows (conflict), return `{ replaced: false, reason: 'external-id-conflict' }`
   - Insert into `collectionQuestions` junction table: `{ collectionId, questionId: inserted[0].id }`

10. Return `{ replaced: true }`

Specifically:
```typescript
import { fileURLToPath } from 'url';
import { dirname, resolve } from 'path';

const __filename = fileURLToPath(import.meta.url);
const __dirname = dirname(__filename);

async function tryLoadLocaleConfig(slug: string): Promise<LocaleConfig | null> {
  const configBase = resolve(__dirname, '..', 'scripts', 'content-generation', 'locale-configs');
  const paths = [
    resolve(configBase, `${slug}.js`),
    resolve(configBase, 'state-configs', `${slug}.js`),
  ];
  for (const path of paths) {
    try {
      const mod = await import(path);
      for (const key of Object.keys(mod)) {
        const val = mod[key];
        if (val && typeof val === 'object' && 'locale' in val && 'externalIdPrefix' in val) {
          return val as LocaleConfig;
        }
      }
    } catch { /* not found */ }
  }
  return null;
}
```
