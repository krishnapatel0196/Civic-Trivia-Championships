**Create `backend/src/services/embeddings/AdvancedDupDetectors.ts`:**

Implement three detector classes, each with a static `detect` method that takes the full question list and returns `AdvancedFlag[]`:

**1. AnswerLeakageDetector (DEDUP-05)**

Detects when one question's text contains another question's correct answer text.

```typescript
static detect(questions: QuestionForDedupFull[]): AdvancedFlag[]
```

Algorithm:
- For each question, extract its correct answer text: `question.options[question.correctAnswer]`
- Normalize both question texts and answer texts (lowercase, collapse whitespace, strip punctuation)
- For each pair (qA, qB): check if normalized qA.text contains normalized qB's correct answer text (and vice versa)
- Only flag if the answer text is >= 4 words long (avoid flagging short common phrases like "the Senate" or "George Washington" which appear naturally in civics questions)
- Severity: 'high' if the full answer appears verbatim in the other question text, 'medium' if a substantial substring (>80% of words) appears
- Evidence: quote the leaked answer text and where it appears in the other question

**2. SourceClusterDetector (DEDUP-06)**

Detects questions from the same source URL whose explanations may reveal each other's answers.

```typescript
static detect(questions: QuestionForDedupFull[]): AdvancedFlag[]
```

Algorithm:
- Group questions by `source.url` (exact match after normalizing trailing slashes)
- For each group with 2+ questions from the same source URL:
  - For each pair in the group, check if qA's explanation text contains qB's correct answer text (or vice versa)
  - Also check if qA's explanation text contains substantial overlap with qB's explanation (>50% shared sentences), indicating they were mined from the same paragraph
  - Sentence splitting: split on `. ` (period + space) for simple but effective segmentation
- Severity: 'high' if explanation directly contains the other's answer, 'medium' if explanations share >50% sentences
- Evidence: quote the shared source URL and the overlapping explanation text

**3. InverseDuplicateDetector (DEDUP-07)**

Detects complementary question pairs where one question provides as context what the other asks for.

```typescript
static detect(questions: QuestionForDedupFull[]): AdvancedFlag[]
```

Algorithm:
- For each question, extract:
  - The correct answer text: `options[correctAnswer]`
  - Key entities/facts mentioned in the question text (simple extraction: look for proper nouns, numbers, dates, named entities that appear in both questions)
- For each pair (qA, qB):
  - Check if qA's correct answer text appears in qB's question text AND qB's correct answer text appears in qA's question text
  - This indicates an inverse: Q1 gives X and asks Y, Q2 gives Y and asks X
  - Example: "What year did [person] take office?" (answer: 1865) and "Who took office in 1865?" (answer: [person])
- Only flag if BOTH directions match (A's answer in B's text AND B's answer in A's text)
- Severity: 'high' if both answers appear verbatim, 'medium' if one is a close substring match
- Evidence: quote both question-answer pairs showing the swap pattern

**Convenience runner:**

```typescript
export function runAllAdvancedDetectors(questions: QuestionForDedupFull[]): AdvancedFlag[] {
  return [
    ...AnswerLeakageDetector.detect(questions),
    ...SourceClusterDetector.detect(questions),
    ...InverseDuplicateDetector.detect(questions),
  ];
}
```

IMPORTANT: These detectors are pure functions operating on in-memory data. They do NOT call any external APIs or database. They run after the embedding-based scan has completed, as an additional analysis pass on the same question set.

The 4-word minimum for answer leakage is critical to avoid false positives. Civics questions naturally share short phrases like "the President", "the Constitution", "United States" - these are not answer leakage.
