<task type="auto">
  <name>Task 2: Create merge-generated-questions.ts utility script</name>
  <files>backend/src/scripts/merge-generated-questions.ts</files>
  <action>
Create a CLI utility script at `backend/src/scripts/merge-generated-questions.ts` that merges the output of `generateQuestions.ts` into the canonical source JSON files.

**Requirements:**

1. **CLI interface:**
   ```bash
   npx tsx src/scripts/merge-generated-questions.ts --input <path-to-generated-json> --collection <slug>
   ```
   Where `<slug>` is one of: `bloomington-in`, `los-angeles-ca`, `fremont-ca`, `indiana` (maps to `indiana-state-questions.json`), `california` (maps to `california-state-questions.json`)

2. **Mapping from collection slug to source JSON file** -- reuse the same JSON_FILE_MAP as generateQuestions.ts:
   ```typescript
   const JSON_FILE_MAP: Record<string, string> = {
     'bloomington-in': 'bloomington-in-questions.json',
     'los-angeles-ca': 'los-angeles-ca-questions.json',
     'indiana': 'indiana-state-questions.json',
     'california': 'california-state-questions.json',
     'fremont-ca': 'fremont-ca-questions.json',
   };
   ```

3. **Merge logic:**
   - Read the generated JSON file (the `--input` path). It has structure: `{ topics: [...], questions: [...] }`
   - Read the canonical source JSON file from `backend/src/data/<filename>`. Same structure.
   - Append all questions from the generated file's `questions` array to the source file's `questions` array.
   - Check for externalId collisions: if any generated question's externalId already exists in the source questions array, SKIP it and log a warning. This prevents double-merges if the script is run twice.
   - Write the updated source JSON back with `JSON.stringify(data, null, 2)` + trailing newline.
   - Log: how many questions appended, how many skipped (duplicate externalId), new total in source file.

4. **Safety:**
   - If `--input` file does not exist, error with clear message.
   - If source JSON file does not exist, error with clear message.
   - If generated questions array is empty, warn but do not error.

5. **No database interaction.** This script only touches JSON files.

**Do NOT:**
- Import any database modules
- Touch the topics array in the source file (leave it unchanged)
- Modify any fields in the generated questions (copy verbatim)
  </action>
  <verify>
1. `npx tsx src/scripts/merge-generated-questions.ts --help 2>&1 || true` -- should print usage or not crash
2. Create a tiny test: create a temp JSON file with 1 question, run the merge against a copy of a source file, verify the question count increased by 1. Clean up after.
  </verify>
