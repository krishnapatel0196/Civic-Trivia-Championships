## Gap Details

### Gap 1: Elaboration Text Field Mismatch (ADMN-03)

**Severity:** Functional bug
**Impact:** Admin cannot see player free-text elaboration in flag review panel. Reasons chips display correctly, but the optional text feedback is invisible.

**Root cause:** Backend admin.ts line 949 sends `elaborationText`, but FlagDetailPanel.tsx line 12 expects `elaboration`.

**Evidence:**
- Backend: `elaborationText: row.elaboration_text` (admin.ts:949)
- Frontend: `elaboration: string | null` (FlagDetailPanel.tsx:12)
- Frontend access: `flag.elaboration && flag.elaboration.trim()` (FlagDetailPanel.tsx:627)

**Fix:** One-line change in backend/src/routes/admin.ts line 949:
```typescript
// Current (broken)
elaborationText: row.elaboration_text,
// Fixed
elaboration: row.elaboration_text,
```
