## E2E Flow

**Player finds and plays an Arizona collection:**

1. Player opens Dashboard → `useCollections` fetches `GET /api/game/collections` → all 26+ active collections returned (including Arizona tier=state, Tucson/Phoenix tier=city) ✅
2. CollectionPicker renders with search input visible above collection list ✅
3. Player types "arizona" → `isFiltering=true` → flat grid shows only "Arizona" card ✅
4. Player types "az" → flat grid shows "Tucson, AZ" and "Phoenix, AZ" (neither "Arizona" nor "Federal" match "az" substring) ✅
5. Player clears input → `isFiltering=false` → grouped view restores with Federal/State/Local tiers ✅
6. Arizona appears under "State", Tucson and Phoenix appear under "Local" ✅
7. Player selects collection and clicks PLAY → `navigate('/play', { state: { collectionId: selectedId } })` ✅

**Flow: Complete — no breaks**
