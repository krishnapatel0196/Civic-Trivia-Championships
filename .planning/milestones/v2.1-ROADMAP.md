## Milestone Summary

**Key Decisions:**

- `runWithinCollectionSemanticDedup()` wired unconditionally into generate-locale-questions.ts — manual scan pass eliminated
- Expiring ratio warning non-blocking (exit 0) — enforcing as block would prevent activating legacy collections; forward-looking nudge only
- Two-query expiry model: `expiringRatioCount` (IS NOT NULL) for ratio; `expiringCount` (90-day window) for netCount blocker
- COLLECTION-PLAYBOOK.md lives in `.planning/` root — cross-phase living document, not per-phase artifact
- Wikipedia API fix: en.wikipedia.org/wiki/ URLs routed through w/api.php extracts API; universal fix applied in fetch-sources.ts
- Washington DC uses city tier; district framing via voice guidance (no district tier in schema)
- Oregon State 7.4% expiring ratio is structural ceiling — 6 viable statewide offices exhausts all viable targets
- Large council budget rule (Biloxi): 2q/ward + 4q/mayor required to reach 15% in 130–170 question pool
- State collection Speaker Pro Tem gap: must be listed by name in locale config or added manually
- `checkAddressPhone` changed to blocking severity (applied during v2.1)

**Issues Resolved:**

- Near-duplicate detection gap: automated (PIPELINE-01) — manual scan pass eliminated from standard workflow
- Expiring ratio enforcement: audit script now warns at <15% (PIPELINE-02)
- Playbook documentation gap: COLLECTION-PLAYBOOK.md captures all learnings, bugs, and patterns (PLAYBOOK-01)

**Issues Deferred:**

- Scaffold Bug 2 persists — generate-locale-questions.ts corrupted by scaffold on every run; revert + manual patch is documented standard pattern; backlogged for dedicated cleanup phase
- Officeholder `expiresAt` not auto-set during generation — every collection required a post-generation targeted pass; backlogged as structured `LocaleConfig` officeholders field with automatic expiry seeding

**Technical Debt Incurred:**

- VERIFICATION.md not created for Phases 59, 60, 62 — activation confirmed via summaries, playbook, and live audit
- 60-02-SUMMARY.md not created — Phase 60 activation confirmed via playbook retrospective and generation report
- Oregon State (7.4%) and Washington DC (9.7%) expiring ratios below 15% target — structural ceilings, documented and accepted
